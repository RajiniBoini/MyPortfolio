Project

VINIMI — Visual Intelligence for Occupational Safety

VINIMI is an end-to-end safety monitoring platform that uses computer vision and vision-language models to automatically detect PPE violations, identify personnel, send real-time alerts, and produce searchable audit logs for compliance.

My role

Lead engineer & designer — owned system architecture, model integration, real-time pipeline, and alerting infrastructure.

Core technologies

  • YOLOv8 (helmet detection), DeepFace (face embeddings), OpenCV
  • Qwen VLM via HuggingFace (vision-language Q&A)
  • FastAPI backend, React + TypeScript frontend
  • MySQL for metadata, Twilio for SMS alerts

Key features

  • Real-time per-frame helmet detection + face recognition (configurable thresholds)
  • Multi-sample worker enrollment with aggregated embeddings
  • Vision-language questions over images/videos with injected detector facts
  • Video analysis with frame sampling and majority-voting for robust ID
  • Cooldown-throttled SMS alerts and audit-ready violation logs

Screenshots

Impact

VINIMI reduces manual inspection overhead, improves incident traceability, and speeds compliance reporting. Typical performance: ~200ms per frame (YOLO + embedding), gallery search <50ms.

Security & privacy

Embeddings are stored instead of raw faces, retention policies and auto-delete options are configurable, and APIs use JWT-based auth with role-based access control.
